일일영양
=======

요약
----
매일 섭취하는 영양제의 목록을 파악 및 관리하고 여러 사람들과 의견을 나누는 안드로이드 앱입니다.

사용 언어
--
- Kotlin

사용 기술
---------
- Firebase(Firestore Database, Storage, Authentication)
- Material CalendarView
- Glide
- Jetpack Navigation
- DataBinding
- LiveData
- Room
- WorkManager
